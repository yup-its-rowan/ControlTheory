Advanced Control Theory

The world of FTC-level control theory is extremely tested. There are plenty of ways to develop your skills as a programmer even further, even though they aren't going to be explained here. However, I will be linking to more prestigous videos and papers for a more diverse glimpse at what you can pursue in the future.

Pure Pursuit

The base of Pure Pursuit is an algorithm that dynamically computes motor powers based on a current location and an intended path. This computation manipulates the robot's movement into a unique, smoothed form of path following. Pure Pursuit defines itself from other path following algorithms by creating a unique “look-ahead point”, and then applying the correct vectors to move towards that point.

Here are some links to reputed pure pursuit algorithms.

  • 11115's Pure Pursuit Algorithm
  • Path Following Video by The Coding Train

Road Runner

Instead of Pure Pursuit's dynamic path following algorithm, Road Runner is a more traditional trajectory generation and motion planning library. Written specifically for FTC, this pre-made library is a more static path-following. Instead of computing motor powers depending on the difference between the target and the current position, Road Runner plans out entire trajectories using parametric equations, and then executes the planned motion.

Guiding Vector Fields

The two methods above are the most commonly used. However, talk of the next generation of path following algorithms leans heavily on Guiding Vector Fields. These fields are a essentially a map of vectors that push a robot towards a general path.
